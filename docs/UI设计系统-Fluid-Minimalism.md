# UI / UX 设计系统：流动极简 + 杂志排版风

> 目标人群：Gen Z / Alpha  
> 关键词：高感知 · 情绪价值 · 呼吸感 · 去卖场化  
> 落地目录：仅 `C:\project\travel-planner`

---

## 1. 定位

**Fluid Minimalism（流动极简）+ Editorial Tech（杂志排版科技感）**

- 不做 OTA「大卖场」堆砌  
- 主路径压缩为 **灵感流 → 日程画布 → 泡泡分账**  
- 微动效 + 可选触觉（`navigator.vibrate`）增强「精致感」

---

## 2. 色彩（Design Tokens）

| Token | Light | Dark | 用途 |
|-------|-------|------|------|
| 电光薄荷 | `#00E8F0` | 同左高亮 | 主强调、渐变起点 |
| 极光洋红 | `#FF2E93` | 同左 | 流向箭头、渐变终点 |
| 燕麦白背景 | `#F3F2F0` | — | 浅色底 |
| 曜石黑背景 | — | `#121214` | 暗色底 |
| 正文 | `#1A1A1E` | `#F4F4F6` | 阅读 |
| 次级 | `#6B6B76` | `#9A9AA8` | 说明 |

主按钮：`linear-gradient(135deg, #00e8f0 → #6c8cff → #ff2e93)`

---

## 3. 质感

- **毛玻璃**：`backdrop-filter: blur(20px) saturate(1.4)` + 半透明表面  
- **1px 浅边框** + 双层阴影（外投影 + 内高光）  
- 卡片悬浮：hover 轻微 `translateY(-2px~-3px)`  
- 圆角：控件 12–16px，大卡片 20–24px  

实现文件：`css/styles.css`（`:root` / `[data-theme="dark"]`）
