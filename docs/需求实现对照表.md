## 结论

- **能在纯前端 + 免费 API 做的主路径：大多已落地或有可用替代。**  
- **明确没做满的：街景 AR、真机位库、真热力图、真社交/电子签、真航司法规库。**  
- 这些需要后端、合规数据源或原生 App，不适合在当前「本地网页、免商用地图」约束里硬做假功能。
