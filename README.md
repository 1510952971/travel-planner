# 旅途 · Fluid Travel

纯前端旅行规划器：多城环线、日程画布、地图选点、预算/行李/光影等。数据默认保存在浏览器 `localStorage`。

## 从 GitHub 克隆（换电脑）

```bash
git clone https://github.com/<你的用户名>/travel-planner.git
cd travel-planner
```

然后按下面「怎么打开」启动。

## 怎么打开

1. 双击 **`start.bat`**（Windows），或在本目录启动任意静态服务器  
2. 浏览器打开 **http://127.0.0.1:8765**（以 start.bat 为准）  
3. 若样式/脚本未更新，用 **Ctrl + F5** 强制刷新

## 摄影能力 = 普通模式（已常驻）

无需再开「摄影模式」开关。顶栏直接有：

**概览 · 日程画布 · 预算 · 泡泡分账 · 行李 · 光影 · 装备 · 镜头 · 搭子 · 备忘**

- 日程：光影色条、时间拨轮、GPS 坐标、跨天拖拽  
- 光影：日出日落/黄金蓝调、云量、银河季、安全提示  
- 装备：称重 / Wh / 航司模板  
- 镜头：FOV + 拨轮  
- 搭子：本机演示板 + TFP 模板  
- 深浅色：侧栏 **◐**  

## 其它功能

扭蛋行程、地图折线、路线优化、天气行李、预算图/AA、撤销、导出 JSON/MD/分享页/摄影志、离线壳等。
